# Command Lists

`/assemble_all`\
``\
``เป็นคำสั่งที่ใช้สำหรับการเรียกทุกคนที่อยู่ใน Voice Channel อื่นๆใน Discord Server นั้นๆ มายัง Voice Channel ที่ผู้ใช้คำสั่งอยู่

`/assemble_category`\
``\
